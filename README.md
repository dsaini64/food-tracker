# FoodSnap – AI-Powered Food Tracking

## Problem
Manual calorie tracking is tedious and inaccurate.

## Solution
An AI-powered app that estimates nutrition from text or photos.

## Tech Stack
- Frontend: React / iOS
- Backend: Flask / Node
- AI: OpenAI Vision / LLM
- Storage: SQLite / Postgres

## Key Challenges
- Prompt reliability
- Handling uncertainty
- UX tradeoffs

## What I’d Improve Next
- Caching
- Model evaluation
- Personalization
